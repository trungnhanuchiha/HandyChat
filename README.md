## Chi tiết thiết kế

### Tổng quan

Server sử dụng Nodejs, giao diện ứng dụng ở client sử dụng java. 

### Tính năng

1. Khả năng chat 1-1
2. Chat nhóm nhiều người
3. voice chat, video call
4. file transfer
5. Tính năng thêm: block user khác, đăng nhập ẩn (với tất cả hoặc với một số user khác)
